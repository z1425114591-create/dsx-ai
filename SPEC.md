# 早读动物园 - 规格说明

## 1. 项目概述

- **项目名称**: 早读动物园
- **类型**: 互动教育/阅读应用
- **核心功能**: 通过检测朗读音量，在屏幕上生成动物emoji，让早读变得更有趣
- **目标用户**: 学生、晨读参与者

## 2. 视觉与渲染规格

### 布局结构
- **顶部**: 标题"早读动物园" + 副标题"一起大声朗读吧！"
- **中间展示区**:
  - 天蓝色渐变背景（模拟天空）
  - 浅绿色地面区域（底部约20%）
  - 动物emoji随机散落分布
- **底部三栏**:
  - 左侧: 当前分贝显示 + 阈值调节滑块
  - 中间: 计时器显示（分:秒）
  - 右侧: 设置按钮 + 暂停/开始按钮

### 颜色方案
- 背景渐变: #87CEEB (天蓝) → #E0F7FA
- 地面: #90EE90 (浅绿) 渐变
- 主色: #4A90D9 (蓝色)
- 警告/停止: #E74C3C (红色)
- 卡片背景: rgba(255, 255, 255, 0.95)
- 文字: #333333

### 字体
- 标题: 楷体, 32px, 加粗
- 正文: 系统无衬线字体, 14-16px

## 3. 功能规格

### 音频检测
- 使用 Web Audio API + navigator.mediaDevices.getUserMedia
- 采样率: 44100Hz
- 计算音量 RMS 值转换为分贝
- 显示范围: 0-120 dB
- 默认阈值: 60 dB

### 动物生成
- 当音量 > 阈值时，按设定时间间隔生成
- 生成速度: 1-60秒可调，默认5秒
- 动物大小: 30-60px 随机
- 位置: 展示区随机坐标（避开地面区域）
- 动画: 淡入 (opacity 0→1, 500ms)
- 动物保留在屏幕上直到手动清除或达到最大数量（50个）

### 计时器
- 格式: MM:SS
- 累计朗读时间（仅在开始状态下计时）
- 暂停时时间保持

### 设置面板
- 点击设置按钮弹出模态框
- 包含:
  - 动物生成速度滑块 (1-60秒)
  - 动物种类多选列表（默认全选）
  - 全选按钮
  - 全不选按钮
  - 保存按钮
  - 重置按钮（恢复默认设置）
- 点击外部或取消按钮关闭

### 数据持久化
- localStorage 保存用户设置:
  - 阈值
  - 生成速度
  - 选择的动物种类

## 4. 交互规格

### 控制按钮
- **开始/暂停**: 切换计时和音频检测
- **设置**: 打开设置面板

### 滑块
- 阈值滑块: 实时显示当前 dB 值
- 速度滑块: 实时显示当前秒数
